ホーム/メソッド

EarthiUs のメソッド

人生のあらゆる場面で使える、共通の心理メソッド群。あなたの今に響くものから始められます。

EarthiUs を生きる、3つの柱

[ GMC IMAGE ]

GMCシリーズ — Great Mother Card

「そのままのあなたで素晴らしい」を、あなた自身の口癖にしていく。完璧な慈愛を持つ"偉大なる母親"が贈る言葉のメソッド群。

大人になってから、もう一度自分の中に「無条件の愛」を作り直すための、144の言葉から始まる体系。

GMCシリーズを見る

  • GMC初級 144 詳しく → 144のアファメーションで、自己肯定の言葉を口癖にする入門ステップ。
  • GMC中級 リラ 詳しく → 関係性の中で、自分の声を見つける中級メソッド（Deep and Calm Meeting）。
  • GMC-Be Proudly 詳しく → 誇りをもって自分を語れるようになる、自己宣言のステージ。
  • GMCティーチャーズスキル 詳しく → GMCを次の世代に伝えるためのティーチング技術。
[ JCR IMAGE ]

JCR — ジュエルキューブリフレクション

色を介して、言葉にならない自分の内側を映し出す。日本初の色彩心理療法として立川智子が体系化したEarthiUs上級メソッド。

医療や教育の現場で驚くべき成果を上げ、雑誌・WEB・ラジオで話題となった、深い自己理解のためのプロフェッショナル・メソッド。

JCRを詳しく見る

[ DIARY IMAGE ]

EarthiUs Diary（Bible）

EarthiUsの哲学を、日々の暮らしに溶かしていく一冊。書くことで、あなたの口癖を、世界の見え方を、少しずつアップデートしていく。

アーシアスの世界観を365日伴走させるためのバイブル。1冊・2冊付きから選べます。
